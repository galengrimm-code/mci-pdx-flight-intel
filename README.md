# MCI - PDX Flight Intelligence

Personal flight intelligence app for tracking door-to-gate travel time, flight pricing, and miles vs cash value.

## Setup

1. Create a Google Sheet
2. Go to Extensions - Apps Script
3. Paste the code from GOOGLE_APPS_SCRIPT.js
4. Run setupSheet function to create tabs
5. Deploy as Web app (Execute as: Me, Access: Anyone)
6. Copy the URL
7. Deploy this app to Vercel
8. Go to Settings and paste your Apps Script URL

## Tech Stack

- React + Vite
- Google Sheets via Apps Script
- Vercel hosting
